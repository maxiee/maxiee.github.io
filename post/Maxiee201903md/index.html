Maxiee 的 2019.03 月度总结

2019-03-29

前言

时间过得飞快，2019 年的第一季度过去了，这一个月里我在技术上学到很多，在生活中也经历了很多，下面是我这个月的收货总结。这个月一共写了 23 篇博客！使出了洪荒之力！三月的我就像一双滑板鞋，在这现实的地上摩擦、摩擦，是爪牙，是魔鬼的步伐。

这还远远不够，还需要继续努力。

React Native 是我今年技术精进的主题。我要把这份热情保持下去。

  1. 使用 Scrapy 和 DeltaFetch 进行增量爬取（翻译）
  2. react-static 静态网站搭建（三）：React 静态博客 Step by Step

写作对学习有很大的促进作用，这点毋庸置疑。

翻译文章的收获

这个月我开始尝试着翻译文章，从中收益良多，我想分享一下翻译文章的收获。

向优秀的人学习

我是一个比较平庸的人，眼界有限，创造不出太多高级东西。

因此，与其浪费时间折腾中华土味，不如看看优秀聪明的人们讲了什么、做了什么。

很感激他们无私奉献的精神，愿意拿出宝贵的时间，悉心将知识总结成文章分享出来。

通过翻译这些人的文章，大大开拓了我的眼界：

  • 有些不太熟练的技术(React Navigation、Redux)，通过翻译文章、学习优秀项目学会如何使用、如何用好了
  • 通过翻译 React Native 的一些开源进展，让我学到了 RN 开源社区的玩法，也知道了社区中的知名开发者。(我已经给 RN 提了两个 PR 了，修了一个 Typo，修了一个 Bug，虽然有一个没合入，但是修复方法被采用了)
  • Scrapy DeltaFetch 这篇绝对 saving me a hundred of hours，天知道基于我之前的那丁点知识能白白浪费多少时间，来实现这个社区中早就有的轮子

如果我没有翻译这些文章，我肯定拿这些时间去写些蹩脚的 SideProject，最终半途而废，还十分沮丧。

所以说，庸人自扰，有些痛苦纯粹是自找的。

英文提升

翻译了 10 篇文章，没想到英文也提升了。

现在看到英文感到很亲切，字里行间都变得可爱了。

之前我尝试过多次用英文记笔记都失败了。

原因就是用英文记下的笔记，等回头看的时候，就好像在看别人写的文章，我还得再在脑子里翻译一遍。

现如今，我再看自己的英文笔记的时候，感觉跟中文差不多了，打眼就能看得进去、就能看懂。

翻译英文文章这件事情好处很多，以后我要坚持下去。

成长

三月给我带来收获最大的不是技术方面，而是个人的成长方面。

我的性格是属于幼稚型的，也十分缺乏社会阅历，从小在父母的保护伞下长大，被过度溺爱。

相对于已婚人士这个身份来说，我实际上是远远不合格的。

三月的我就像一双滑板鞋，在这现实的地上摩擦、摩擦，是爪牙，是魔鬼的步伐。

我感觉我的心里年龄一下成长了好几岁，快速成长，变得成熟了许多。

我是幸福的

一直以来我就是个吐槽精，屁大点事都要生气、情绪都会波动。

还会摆出一种怨天尤人的样子，到处散发负能量。

也不知怎么的，一下子我就懂得了一个道理：

在这个世界上，家庭不完美的人有很多，填不饱肚子的人有很多，没有知识改变命运的人有很多，养不起孩子的很多，得了疾病的人有很多，犯了不可弥补的错误的人有很多……

更让我感到极度内疚的是，这些人中好多都没有抱怨，甚至都保持着开朗乐观，积极向前、拥抱生活。

生活的重担与挫折会压得他们透不过起来，他们也会时常叹息，但更多的是，他们会挺起胸膛勇敢地生活。

满身伤痕的人都没有抱怨，我因为一点小事就开始矫情，羞愧极了。

过去的错误

我在过去犯过很多很多错误，每到夜里这些过错便涌上心头，折磨着我。

世界上没有后悔药，结局已定，没有挽回的机会了。

许许多多个夜晚，我都是在悔恨与无奈中无眠。

我不想说什么鸡汤，鸡汤只是一些糖丸，能让心里暂时舒服些。等糖分解了，这些悔恨的小鬼又爬上心头了。

我想说的是，当折磨的次数成百上千之后，摆在面前的选择其实很清晰：

  • 要么生，要么死

我虽然犯过很多错误，但是还不至于要用死来弥补的地步。

那摆在面前就剩下一条路，既然要活，就要好好活。

既然犯了错，就要接受现实，就要承担责任。

毕竟自己犯了错，别人要说自己不是人，那不是别人的过错，自己确实在这件事情上不是人。

我所要做的是，改过自己的过错，弥补自己造成的伤害，尽力而为。

同时，自己的生活也要负责地过。

负能量与焦虑

最近在看房、学车，同时工作上也正在经历较大的波动，要说压力还是不小的。

我感到很焦虑。

但是我不能让它蔓延，我要控制它。

负能量也是这样，负能量是能上瘾的，因此我也要驱散它。

我希望自己做一个开朗的人、乐观的人。

我希望自己做一个欢笑时就大声欢笑，哭泣时就放声大哭，乐观豁达的人。

简单平凡

我老想着写程序挣钱，但现实告诉我，我其实不擅长干这个。我是一个爱学习的人，虽说学的不太好，但是好这一口。

我现在能够理解孔乙己了，同时感觉自己也会成为这么一个百无一用的书生。

如果我跟自己和解，接受这份简单、平凡的人生，我想我的世界会更加多姿多彩。

当然，迫于生活的压力，也需要去跳槽、接着写程序挣钱，但我不再是我的敌人。

多姿多彩的生活

夜里我从五道口坐车往家走，看着纷繁的街道，看着形形色色的车辆，看着匆匆的人群。

每一盏灯都照亮一个梦，每一个脚步代表一种生活。

我突然感到，这些我习以为常的东西变得新奇起来，暗淡无光的生活变得多姿多彩。

我有点懂得为什么要有文学，为什么要有艺术，为什么要浪漫。

人是主观的动物，是情感的动物，人不是机器，也成不了机器。

四月

想到我刚来北京的时候，是带着一个睡袋来的，连枕头都没有。

一切都不确定、一切都那么陌生、一切都新奇有趣。

那时我把自己的心完全打开，拥抱一切。

那段时光虽然很苦，但是回忆起来很美，美的发光。

我想那才是人生该有的活法。

四月，希望我能找回那个我。
