Maxiee 的 2019.04 月度总结

2019-05-14

已经是 5 月中旬，才想起来 4 月份的总结还没有写，赶忙补上。4 月份工作和生活上事情都很多，用于学习的业余时间相对少了一些，不过收获还是令人满意的。

React Native 学习

本月写了 9 篇关于 RN 的博客，他们是：

  • React Native JSI 尝鲜
  • React Native 的新架构介绍（三）：Fabric 和 TurboModules
  • React Native Redux、redux-persist、immutable.js 结合实践
  • 如何编写 React Native 的 CxxModule
  • React Native 的新架构介绍（二）：JSI 和 JSC
  • React Native 的新架构介绍（一）：React 和 CodeGen
  • React Native 0.59 版本发布了
  • React Native 代码阅读（十四）：UIManagerModule 与 ReactNativeRenderer
  • React Native Bundle 懒加载(翻译)

为了巩固对 RN 的学习，我还开发了一个个人聊天助手形式的 RN App，对 React Native 和 Redux 都进行了深入实践，对 RN 的掌握越来越熟练。

未来我可能不会再安排这么多时间来翻译文章了，有其它优先级更重要的事情要去做。

我希望能在网站中开辟出一个 RN 专区，做成类似湾区评论的形式，只传递链接和评论即可。这个会在下文中介绍。

React Native 原理串讲

五月份我在团队内进行了一次分享，主题是 React Native 实现原理。

从 React、React Native 的演进过程讲起，逐渐深入到实现原理，并介绍了 Fabric 重构，最终分享了如何通过参与开源来进行自我提升。

这算是对这几个月来的 React Native 学习做一次比较好的总结。

React Native 非常庞杂，代码量非常大，就像一座庞大的城市。学习它的原理就像一位观光客来到这座城市，尽管有别人编写的地图、指南，但是最终都需要自己去熟悉街道、建筑。

当自己对这座城市愈发熟悉，就有一种轻车熟路的感觉。并且，尽管自己只是在地面上行走，但是脑海中就像有一架无人机一样，能够对这座城市进行宏观上的鸟瞰。

讲解如此复杂的东西，关键在于如何能用最简单的话语，把最核心的东西提炼出来。这个过程前后进行了约一个月时间。这种思考的方式非常好，对个人的思维能力提升很大。

五月份不再有这么多时间学习 RN 了，我希望每周能抽出一个时间来关注下它的进展。

我对这项技术还是非常看好的。

maxiee.github.io 升级

maxiee.github.io 稳定运行了几个月时间，我想又到了升级的时候了。主要有以下想要升级的地方：

  • 主题：我想改成向 Medium 那样的设计样式
  • 统计：我想使用 LeanCloud 开发自己的数据统计后台，积累开发数据统计平台的经验
  • 开发一个 RN 主题子站，这个还在构思当中
  • 目前 Markdown 的格式需要调整，最好把 meta data 放在 md 文档内部

总之这是个大活。(现在已经五月中旬了，我还在拖延着呢！)

展望五月(其实今天是 5 月 14 号！)

那我就展望下半月吧……

五月份我在工作中正在参与一项动态化相关的项目，非常有挑战性，也很有意思，它占据了我的主要精力。

我在五月份的业余精力都投入在编写一个知识管理软件上面。

等知识管理软件编写地基本能用的时候，我会立刻再去开发一个个人任务管理规划软件。

因为知识管理和任务管理，这两个是折磨我近十年的终极问题，让我持续焦虑了十年。

我尝试过各种方法和工具，试图来应对它，但最终都无法战胜焦虑。

我发现我没法逃避它，因此在经历了多次放弃之后，这次我又鼓起勇气，向它发起挑战了。

五月底又要开始学车，六月份所有的周末不知是否还有精力学习。

总结

2019 年已经度过 36.7%，不过才五月而已，新年刚刚过去才两个来月，但 19 年都过去三分之一多了。

我感觉随着年龄的增大，半年的重量越来越大。

以前上学的时候不觉得半年有什么，感觉不过就是度过一个学期，然后又放假了而已。

可现在呢，事赶事，在这个年龄段，结婚、领证、买房、买车、生小孩、照顾父母、处理亲戚关系等等。

现在回头看看，上学那点压力真是不算啥。

我想现在也是如此，等我七八十岁，走也走不动，时间掐着秒表来过的时候，也会怀念三十岁的时候，充满了精力，有那么丰富的人生去体验，一切都充满可能性。

积极乐观地活在当下！
