Dukto -- 跨平台局域网传输软件

2016-12-04

Dukto 是一款跨平台局域网传输软件. 它具有如下特点:

  • 开源
  • 跨 Windows, OS X, Linux, iOS, Android 等平台的局域网传输软件
  • 不需要连接服务器, 不需要连接因特网
  • 无需配置, 简单易用
  • 传文件或者传文本

Dukto 的网站首页.

安装

Windows

Dukto 为 Windows 平台提供了现成的二进制. 下载地址, 其中:

  • DuktoR6-Portable.zip 是绿色版
  • DuktoR6-Setup.exe 是安装版本

Fedora 和 Debian 和 Ubuntu

Dukto 为这几个发行版也提供了现成的二进制. 下载地址 跟上面相同, 找到对应 .rpm 或 .deb 包安装.

Arch Linux

对于 Arch Linux 发行版, 在 AUR 中有第三方打包的 Dukto.

使用

Dukto 安装成功后打开的界面如下:

WindowsArchLinux

由于在两个平台上的界面是完全一样的, 在这里就以 Windows 平台为例, 演示如何向另一台 Linux 电脑传送数据.

文件发送

在上面的界面中, 点击列表下方的 Maxiee (Linux 电脑), 会弹出如下界面:

我想将一个文件发送至另一台电脑, 因此我点击 Send some files.

之后弹出文件选择对话框, 选中发送文件, 点击确定.

此时程序界面会更新, 告诉我文件已经发送给了我的小伙伴:

而在发送的另一侧, 也就是那台 Linux 电脑上, 程序的界面也自动发生了变化, 自动进入到了 Tab 页面:

从中可以看出, 文件已经成功接收.

文本发送

文本发送只需点击 Send some text, 在弹出界面的编辑框中输入待发送文本即可.

默认保存路径

最后一点要备注的是, 在设置中可以修改文件的默认保存路径.
