有偿开发服务

我是一名资深的软件开发工程师，具有多个大型商业项目的从零到一的成功经历。

迫于生活的压力（买房、生子、赡养老人），仅靠工资难以维持生存，我打算利用我的个人业余时间提供有偿开发服务。

对于个人或创业团队，如果你们有一个好点子（给媳妇或子女开发一款 APP，或者有一个可以改变世界的奇妙点子）但是缺少软件工程师来实现的话。如果在我的能力范围之内，我可以为你们提供开发服务。

关于我的个人能力，可以参见我的「博客文章」和「关于」。对于中小型的程序，我有信心能够提供专业的质量水平，同时相较于外包团队提供更有吸引力的服务价格。

技能

我掌握以下技能，可以胜任以下类型程序的开发工作：

Android 原生 APP 开发

我的本职工作是资深 Android 开发工程师，作为负责人与核心开发者开发过多款商业 APP 的从零到一。

如果你想开发一款 Android 应用，并且具备精美、流畅的视觉效果，我有信心能够干好！

React Native APP 开发

我也有丰富的 React Native 开发经验，并在大型商业 APP 中的成功开发经验。

React Native 是一个跨端开发技术，能够同时开发 Android、iOS 应用。

并且相较于原生开发，React Native APP 的开发周期更快，更加适合于创业团队。

如果你想开发一款 Android、iOS 都支持的应用，并且想要短时间上线，我想我可以帮你用 React Native 技术实现它！

网站开发

如果你想开发一个中小型的网站（产品展示、个性化博客），我可以提供帮助。

本站就是我完全独立开发出来的，它是一个静态网站，不需要高昂的服务器维护成本，只需要域名的开销。

我擅长静态网站开发技术，以及 React 网站开发技术，均为目前前端界比较前沿的开发技术。

能够开发出优美、时髦的网站。

小游戏开发

如果是你一名设计师，有一个绝妙的点子，想开发一个小游戏（Android、iOS 游戏，非微信小游戏）。

如果你能提供视觉、音频素材的话，我可以通过代码将它们组装到一起，让点子动起来！

底层技术可以选用 Unity 或者 Cocos2D-x。

爬虫、数据采集项目开发

如果你自动化需要获取某些网络数据，并进行汇总、分析能力的话，我可以为你开发专属爬虫、数据分析软件。

我自己有微博、AppStore 以及许多普通网站的爬取经验，并具备一定的反爬能力。

同时我也能提供可视化的 Web 页面，或者 GUI 展示页面，能够让你方便地检索爬取结果。

科学计算软件开发

我在硕士期间研究方向为统计信号处理领域的卡尔曼滤波算法。

能够编写各种（线性、非线性）卡尔曼滤波器相关算法，用于预测。

同时也能够编写常见的矩阵运算、统计模型相关算法。

对于基本的视觉处理（目标识别、定位）也能编写实现。
